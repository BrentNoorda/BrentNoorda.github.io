JSE_PACK_SECODES (off, on if JSE_MIN_MEMORY is on)

 

The bytecodes are packed, conserving memory at the cost of execution speed. Bytecodes can only be run on a ScriptEase application with the same setting for this define as the one that created the bytecodes.
