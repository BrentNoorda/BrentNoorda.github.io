CHARACTER SET CUSTOMIZATIONS

 

Choose a character set. These are exclusive options, uncomment the define for the character type you wish to use. JSE_ASCII stands for an ASCII character set build and is the default. JSE_UNICODE and JSE_MBCS (multibyte character set builds) are the other two possibilities.

 

JSE_ASCII (on)

JSE_UNICODE (off)

JSE_MBCS (off)

JSE_USER_DEFINED_MBCS (off)
