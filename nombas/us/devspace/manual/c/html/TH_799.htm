Dos.inportw()

syntax:

Dos.inportw(portid)

where:

portid - port from which to read.

 

return:

number - 16 bit word of data from a hardware port.

 

description:

Read a word (16 bit) from hardware port: portid. Value read is unsigned (not negative).

 

see:

Dos.inport()
