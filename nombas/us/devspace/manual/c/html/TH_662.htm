Clib.isxdigit()

syntax:

Clib.isxdigit(chr)

where:

chr - a character, a single character string.

 

return:

boolean - true if chr is in: 0-9, A-F, or a-f.

 

description:

Returns true if chr is a hexadecimal digit in one of the following sets of characters: 0-9, A-F, or a-f.
