JSE_ALIGN_DATA (off)

 

Some systems require structures to have their members aligned and thus this flag is on for them. It is off for most systems. By not defining it, the default, ScriptEase chooses the right value for your system. However, you can always force it to be on. This will always work, but use more memory if it did not need to be used.
