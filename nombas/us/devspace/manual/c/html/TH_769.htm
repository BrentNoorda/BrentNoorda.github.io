Date setMonth()

syntax:

Date.setMonth(month[, day])

where:

month - a month in a year.

 

day - a day in a month.

 

return:

number - time in milliseconds.

 

description:

This method sets the month, as a number from 0 to 11, of a Date object to the parameter month. January is 0, and December is 11.

 

The parameter day is the same as for Date setDate().
