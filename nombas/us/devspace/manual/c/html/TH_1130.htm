SimpleDataset.caseSensitive

syntax:

SimpleDataset.caseSensitive

description:

Boolean value indicating whether or not find calls are case sensitive. By default, searches are not case sensitive.

 

see:

#include <smdtset.jsh>, SimpleDataset find() with template, SimpleDataset find() with clause

 

example:

// turn on case sensitivity

// for SimpleDataset searches

ds.caseSensitive = true;
