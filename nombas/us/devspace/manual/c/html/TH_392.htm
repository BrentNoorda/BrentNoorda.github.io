Routines for diagnostic/error

 

Clib.clearerr()

Clears end-of-file and error status for a file.

Clib.errno()

Returns value of error condition.

Clib.ferror()

Tests for error on a file stream.

Clib.perror()

Prints an message describing error in errno.

Clib.strerror()

Gets a string describing an error number.
