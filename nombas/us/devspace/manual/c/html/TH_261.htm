Configuring A jseopt.h File

 

ScriptEase ISDK uses a header file called jseopt.h that specifies the compiler options that should be turned off/on when ScriptEase libraries are built. All ScriptEase ISDK applications need a jseopt.h file to specify the various options that are desired for that particular application.

 

New with Version 5.0, you can use theSelib Assistant to configure the jseopt.h file.
