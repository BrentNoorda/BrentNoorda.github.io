Clib.isalnum()

syntax:

Clib.isalnum(chr)

where:

chr - a character, a single character string.

 

return:

boolean - true if chr is in: A-Z, a-z, or 0-9. Else false.

 

description:

Returns true if chr is a character in one of the following sets: A-Z, a-z, or 0-9.
