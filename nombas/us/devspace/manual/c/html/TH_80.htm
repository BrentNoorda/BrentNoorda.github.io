CSTRING (#define JSE_CSTRING_ALL)

 

The CString object provides an object that acts much like a string in the C language. Individual elements may be accessed with str[index] syntax, and pointer-like arithmetic is available (e.g. index = str1 - str2). The Clib object is aware of the CString object and will use it if available..
