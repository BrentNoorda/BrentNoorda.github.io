Dos.outportw()

syntax:

Dos.outportw(portid, value)

where:

portid - port to which to send value.

 

value - a 16-bit word of data to send to the port identified by portid.

 

return:

void.

 

description:

Write a 16-bit word value to hardware port: portid.
