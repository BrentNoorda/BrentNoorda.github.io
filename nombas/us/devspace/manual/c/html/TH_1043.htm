GD getInterlaced()

syntax:

gd.getInterlaced()

return:

boolean - Whether this image is interlaced.

 

description:

If the current image has the interlace flag set, then this method returns true. Otherwise, it returns false.

 

see:

#link <gd>, GD interlace()
