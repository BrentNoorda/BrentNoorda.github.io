Date setMinutes()

syntax:

date.setMinutes(minute[, second[, millisecond]])

where:

minute - a minute in an hour.

 

second - a second in a minute.

 

millisecond - a millisecond in a second.

 

return:

number - time in milliseconds.

 

description:

This method sets the minute, as a number from 0 to 59, of a Date object to the parameter minute. The first minute of an hour is 0, and the last is 59.

 

The parameter second is the same as for Date setSeconds().

 

The parameter milliseconds is the same as for Date setMilliseconds().
