JSE_PACK_OBJECTS (off, on is JSE_MIN_MEMORY is on)

 

Objects are packed to use the minimum memory. This setting is incompatible with JSE_ALIGN_DATA. Turning this on saves significant memory, but on many systems brings an equally significant loss in performance when dealing with objects.
