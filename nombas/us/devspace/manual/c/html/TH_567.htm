Buffer equal()

syntax:

buffer.equal(buffer2)

where:

buffer2 - buffer to compare against.

 

return:

true if two buffers are equal, else false

 

description:

This function is identical to calling Buffer.equal(buffer,buffer2).

 

see:

Buffer(), Buffer equal(), Buffer.compare (), Buffer compare()

 

example:

// The following code:

var bufa = Buffer("ab");

var bufb = new Buffer("ab\0");

var cmp = bufa.equal(bufb)

// will set the variable cmp to be false
