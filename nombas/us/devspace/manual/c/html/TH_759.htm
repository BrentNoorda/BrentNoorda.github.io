Date getUTCMilliseconds()

syntax:

date.getUTCMilliseconds()

return:

number - a millisecond in a second.

 

description:

This method returns the UTC millisecond, as a number from 0 to 999, of a Date object. The first millisecond in a second is 0, and the last is 999.
