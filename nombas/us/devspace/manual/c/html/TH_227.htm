FLAGS

 

The flags parameter is some or all of the following values, |'ed together:

SE_NO_INHERIT

SE_NO_LIBRARIES

SE_NEW_GLOBALS

SE_CALL_MAIN

SE_FUNCS_ONLY

SE_EXIT_LEVEL

SE_NEW_DEFINES

SE_NO_OLD_DEFINES

SE_REPORT_ERRORS

SE_INFREQUENT_CONT

SE_START

SE_CONSTRUCTOR

SE_NAMED_PARAMS

SE_INIT_IMPLICIT_THIS

SE_INIT_IMPLICIT_PARENTS

SE_DEFAULT
