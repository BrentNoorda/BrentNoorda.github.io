Clib.rmdir()

syntax:

Clib.rmdir(dirpath)

where:

dirpath - directory specification to delete.

 

return:

number - 0 on success, else -1.

 

description:

This method deletes the directory specified by the parameter dirpath.

 

see:

Clib.mkdir(), Clib.remove()
