Clib.rename()

syntax:

Clib.rename(oldFilename, newFilename)

where:

oldFilename - current name of file on disk to be renamed.

 

newFilename - new name for file on disk.

 

return:

number - 0 on success, else non-zero.

 

description:

This method renames oldFilename to newFilename. Both oldFilename and newFilename are strings. Returns zero if successful and non-zero for failure.
