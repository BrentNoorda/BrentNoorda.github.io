Date setSeconds()

syntax:

date.setSeconds(second[, millisecond])

where:

second - a second in a minute.

 

millisecond - a millisecond in a second.

 

return:

number - time in milliseconds.

 

description:

This method sets the second, as a number from 0 to 59, of a Date object to the parameter second. The first second of a minute is 0, and the last is 59.

 

The parameter milliseconds is the same as for Date setMilliseconds().
