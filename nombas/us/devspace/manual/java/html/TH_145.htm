FLAGS

 

The flags parameter is any of the following values, |'ed together:

SE.NO_INHERIT

SE.NO_LIBRARIES

SE.NEW_GLOBALS

SE.CALL_MAIN

SE.FUNCS_ONLY

SE.EXIT_LEVEL

SE.NEW_DEFINES

SE.NO_OLD_DEFINES

SE.REPORT_ERRORS

SE.INFREQUENT_CONT

SE.START

SE.CONSTRUCTOR

SE.NAMED_PARAMS

SE.INIT_IMPLICIT_THIS

SE.INIT_IMPLICIT_PARENTS

SE.DEFAULT
