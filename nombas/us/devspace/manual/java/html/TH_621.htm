Clib.srand()

syntax:

Clib.srand(seed)

where:

seed - number with which to seed a random number generator.

 

return:

void.

 

description:

This method initializes a random number generator using the parameter seed. If seed is not supplied, then a random seed is generated in a manner that is specific to different operating systems. Use this method first when generating a sequence of random numbers.
