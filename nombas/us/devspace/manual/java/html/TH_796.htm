RegExp source

syntax:

regexp.source

description:

The regular expression pattern being used to find matches in a string, not including the attributes.

 

Read-only property. Use RegExp compile() to change.

 

see:

Regular expression syntax

 

example:

var str = "one tao three tio one";

var pat = /t.o/g;

pat.exec(str);

   // pat.source == "t.o"
