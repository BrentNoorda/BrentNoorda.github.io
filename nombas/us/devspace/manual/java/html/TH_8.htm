seCreateFiber

 

syntax:

   SEContext

SEContext.seCreateFiber();

where:

None

 

return:

An SEContext, suitable to be used in further ScriptEase API calls.

 

description:

Create a fiber context. Chapter XIII is devoted to the topic of multithreading and fiber contexts. See it for full details.

 

see:

seCreateContext, seDestroyContext
