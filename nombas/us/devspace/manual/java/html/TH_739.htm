Math.SQRT1_2

syntax:

Math.SQRT1_2

description:

The number value for the square root of 2, which is represented internally as approximately 0.7071067811865476. The value of Math.SQRT1_2 is approximately the reciprocal of the value of Math.SQRT2.

 

example:

var n = Math.SQRT1_2;
