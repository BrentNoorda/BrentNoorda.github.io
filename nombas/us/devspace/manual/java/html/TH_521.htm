Clib.remove()

syntax:

Clib.remove(filename)

where:

filename - the name of the file to delete from a disk.

 

return:

number - 0 on success, else non-zero.

 

description:

Delete a file with the filename provided.

 

see:

Clib.rename(), Clib.fopen()
