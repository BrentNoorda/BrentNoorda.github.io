Why These Study Guides?

These are study guides created for the Human Anatomy Laboratory course taught by Dr. James Smiley for the Fall 2015 session taken through UC Berkeley Extension at Merritt College.

I make no guarantee that everything on these slides is correct and complete. In fact, I think I can guarantee that Dr. Smiley’s tests will include some items that are not on these slides, and that some terms on these slides will be ones that Dr. Smiley considers incorrect or outdated.

Feel free to use these slides to learn. It worked for me. If you copy these slides to improve them, all I ask is that you send me a link so I can add a pointer here to your new-and-improved learning tools.
